# 编程工具模块

## 目录说明
本目录包含在线编程工具和WebAssembly编译相关文件。

## 包含文件
- online_programming.html - 在线编程工具页面
- compiler/ - WebAssembly编译功能目录
  - package.json - 依赖配置文件
  - server.js - 编译服务器

## 功能描述
1. **在线编程**：提供C++代码编辑和测试环境
2. **WebAssembly编译**：支持在线C++代码编译和执行
3. **代码测试**：实时测试代码运行结果

## 使用说明
1. 打开 online_programming.html 文件使用在线编程功能
2. compiler目录包含编译服务器代码，需要Node.js环境运行
