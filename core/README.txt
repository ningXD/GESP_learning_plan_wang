# 核心功能模块

## 目录说明
本目录包含GESP C++学习计划的核心功能文件。

## 包含文件
- planTable.html - 8周学习计划表，包含详细的学习内容和时间安排
- git_version.txt - Git版本信息文件

## 功能描述
1. **学习计划表**：提供详细的8周学习计划，包括每周的学习主题、核心知识点和真题实战
2. **笔记功能**：支持添加文本、图片和思维导图笔记
3. **笔记转PPT**：将学习笔记转换为演示文稿
4. **学习日历**：可视化学习进度和时间安排

## 使用说明
直接打开 planTable.html 文件即可使用完整的学习计划系统。
