1. 项目概述

GESP C++学习计划系统是一个专为C++学习者设计的在线学习平台，包含学习计划、消课系统、线上编程等功能模块。系统支持三种用户角色：学生、教师和管理员，每种角色拥有不同的权限。

2. 系统架构

2.1 前端

前端使用HTML、CSS和JavaScript构建，包含以下主要文件：

  • login.html - 登录和注册页面
  • core/study_plan.html - 学习计划表页面
  • course/course_system.html - 消课系统页面
  • programming/online_programming.html - 线上编程页面

2.2 后端

后端使用Flask框架构建，包含以下主要文件：

  • app.py - 应用主文件
  • models.py - 数据模型文件
  • routes/ - 路由目录，包含认证、笔记和用户管理路由

2.3 数据库

使用SQLite数据库（默认），包含以下主要表：

  • users - 用户表，存储用户信息和权限
  • notes - 笔记表，存储用户笔记

3. 核心功能

3.1 学习计划表

学习计划表是系统的核心功能，提供8周冲刺学习计划，包含：

  • 学习周期表：展示每周的学习主题和日期
  • 核心知识点：详细列出每周需要掌握的知识点
  • 真题实战：提供历年真题和重点题目
  • 常用代码模板：提供编程常用的代码模板
  • 笔记功能：支持添加、查看和搜索笔记

3.2 消课系统

消课系统面向教师和管理员，包含：

  • 学员管理：添加、删除学员信息
  • 上课记录：添加上课记录
  • 消课记录：查看所有消课记录
  • 管理员设置：管理教师账户权限

3.3 线上编程

线上编程功能提供编程练习环境，支持：

  • 代码编写：在线编写C++代码
  • 代码运行：运行代码并查看结果
  • 练习题目：提供编程练习题目

4. 权限控制

4.1 学生端

  • 可以访问学习计划表
  • 可以使用线上编程功能
  • 看不到消课系统

4.2 教师端

  • 可以访问学习计划表
  • 可以使用线上编程功能
  • 可以使用消课系统
  • 看不到管理员设置

4.3 管理端

  • 可以访问所有功能
  • 可以管理用户账号和权限
  • 可以访问管理员设置

5. 使用指南

5.1 登录系统

  1. 打开 login.html 页面
  2. 输入用户名和密码
  3. 点击登录按钮

5.2 注册账号

  1. 打开 login.html 页面
  2. 点击"立即注册"链接
  3. 填写用户名、密码和邮箱
  4. 选择用户角色（学生/教师）
  5. 点击注册按钮

5.3 启动服务器

  1. 进入 backend 目录
  2. 运行 python app.py 命令
  3. 服务器默认运行在 http://127.0.0.1:5000
