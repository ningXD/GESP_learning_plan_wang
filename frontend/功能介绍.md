# 前端功能介绍

## 项目概述
前端部分包含了GESP C++学习计划系统的用户界面，主要分为登录、学习计划、消课系统和线上编程等功能模块。

## 文件结构

### 根目录文件
- **auth/login.html** - 登录和注册页面，支持用户登录和注册功能，包含角色选择（学生/教师）

### core 目录
- **study_plan.html** - 学习计划表页面，展示8周冲刺学习计划，包含学习周期表、代码模板、笔记功能等

### course 目录
- **course_system.html** - 消课系统页面，包含学员管理、上课记录、消课记录和管理员设置功能

### programming 目录
- **online_programming.html** - 线上编程页面，提供编程练习环境

### materials 目录
- **real_test_question_materials/** - 存放历年真题材料，包含2023年9月至2025年9月的真题PDF文件

### tests 目录
- **test_features.html** - 功能测试页面，用于测试系统功能
- **test_all_features.html** - 全功能测试页面，用于测试系统的所有功能

## 功能说明

### 登录功能
- 支持用户登录和注册
- 注册时可选择角色（学生/教师）
- 登录成功后根据角色跳转到相应的功能页面

### 学习计划表
- 展示8周冲刺学习计划
- 包含学习周期表、核心知识点、真题实战与重点
- 提供常用代码模板
- 支持添加、查看和搜索笔记
- 根据用户角色显示或隐藏消课系统按钮

### 消课系统
- 学员管理：添加、删除学员信息
- 上课记录：添加上课记录
- 消课记录：查看所有消课记录
- 管理员设置：只有管理员可以访问，用于管理教师账户权限

### 线上编程
- 提供编程练习环境
- 支持代码编写和运行

## 权限控制
- **学生端**：只能访问学习计划表和线上编程
- **教师端**：可以访问学习计划表、线上编程和消课系统
- **管理端**：可以访问所有功能，包括管理员设置
